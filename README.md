# 学生积分管理系统

基于军衔等级制度的学生积分管理平台，参考Class Dojo功能架构开发。

## 功能特性

### 🎯 核心功能
- **军衔等级体系**: 采用军衔等级制度（列兵到大校）
- **积分管理**: 支持学业表现、课堂纪律、课外活动等维度评价
- **双端设计**: 教师端和学生端不同权限界面
- **数据分析**: 多维度数据统计和可视化展示

### 👨‍🏫 教师端功能
- 学生档案管理
- 积分授予/扣除（个人和批量）
- 军衔晋升追踪
- 数据统计与分析
- 通知与反馈系统

### 👨‍🎓 学生端功能
- 个人积分和军衔展示
- 积分历史记录
- 班级排行榜
- 成就徽章系统
- 积分兑换系统

### 🏆 扩展功能
- 成就徽章系统
- 积分兑换商城
- 实时通知推送
- 多语言支持

## 技术架构

### 前端技术
- **框架**: React 18 + TypeScript
- **样式**: Tailwind CSS 3
- **状态管理**: Zustand
- **图表**: Recharts
- **通知**: Sonner
- **图标**: Lucide React

### 后端服务
- **平台**: Supabase (BaaS)
- **数据库**: PostgreSQL
- **认证**: Supabase Auth
- **存储**: Supabase Storage

### 部署平台
- **前端**: Vercel
- **后端**: Supabase云服务

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置环境变量
复制 `.env.example` 为 `.env` 并填写你的Supabase配置：
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问应用
打开浏览器访问 `http://localhost:5173`

## 军衔等级体系

| 军衔 | 等级 | 所需积分 | 描述 |
|------|------|----------|------|
| 列兵 | 1 | 0 | 起始军衔，表现良好的学生 |
| 上等兵 | 2 | 100 | 表现优秀的学生 |
| 下士 | 3 | 300 | 表现突出的学生 |
| 中士 | 4 | 600 | 表现卓越的学生 |
| 上士 | 5 | 1000 | 表现优异的学生 |
| 少尉 | 6 | 1500 | 表现出色的学生干部 |
| 中尉 | 7 | 2100 | 表现杰出的学生干部 |
| 上尉 | 8 | 2800 | 表现卓越的学生干部 |
| 少校 | 9 | 3600 | 表现优异的学生领袖 |
| 中校 | 10 | 4500 | 表现杰出的学生领袖 |
| 大校 | 11 | 5500 | 表现卓越的学生领袖 |

## 积分类别

- **学业表现**: 考试成绩、作业质量等
- **课堂纪律**: 参与度、纪律表现等
- **课外活动**: 社团活动、竞赛参与等
- **行为表现**: 品德行为、同学关系等
- **班级贡献**: 为班级做出的特殊贡献
